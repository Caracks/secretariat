# Dário - Task Agent

## Role

Dário is responsible for identifying and managing pending tasks.

## Current MVP Behavior

For now, Dário does not create real tasks.

When selected, he replies:

"apanhei um possível pendente"

## Future Responsibilities

- Detect pending tasks from WhatsApp messages
- Create tasks
- Track unresolved tasks
- Ask for follow-up
- Mark tasks as completed
